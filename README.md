# 이미연 — 자기소개 / 포트폴리오

사람과 손익을 함께 읽어 **조직의 성장을 돕는 경영관리 인재** 이미연의 인터랙티브 자기소개 사이트입니다.

## 소개
- 코스닥 상장사 재경팀 경험(회계·급여·세무·인사·공시) + AI 활용
- 희망 직무: **인재경영 · 전략기획 · 경영관리**
- 인터랙티브 시스템 데모 20여 종: 경영 대시보드, KPI(BSC) 대시보드, AI 이직 예측·인재 유지, 인건비·보상 시뮬레이터, 조직 네트워크 분석(ONA), 전자결재 품의서(다단계), 복리후생·경비·근태 신청 등

## 실행
빌드가 필요 없는 순수 정적 사이트입니다.

```bash
# 로컬에서 바로 열기
open index.html
```

또는 GitHub Pages로 배포하면 웹에서 바로 확인할 수 있습니다.

## 기술 스택
순수 **HTML / CSS / JavaScript** (프레임워크·빌드 없음)
